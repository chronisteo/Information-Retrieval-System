# Greek Parliament Speech Analysis

This project implements an **Information Retrieval and Text Mining system** for analyzing speeches from the **Greek Parliament (1989–2020)**.

The system preprocesses political speech data, extracts important keywords, and identifies major thematic topics using **TF-IDF** and **Latent Semantic Indexing (LSI)**. It also allows users to explore how topics evolve over time.

The project includes an **interactive web interface** built with **PyWebIO**, where users can analyze speeches, extract keywords, and visualize topic similarities.

## Technologies

* Python
* spaCy (Greek NLP)
* scikit-learn
* pandas
* PyWebIO
* Plotly

## Main Features

* Text preprocessing and lemmatization
* Keyword extraction from speeches
* Topic modeling with LSI
* Topic evolution visualization
* Interactive web interface
